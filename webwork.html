Blog Post Project

This project was to deepen my understanding of Node.js, backend development, and working with databases. I used JavaScript and Node.js to create a basic REST API that handles creating and retrieving blog posts. I connected a SQL database to store the posts so that the data would persist even after the app was closed. To test and debug my API routes, I used Postman, which helped me better understand how HTTP requests like GET and POST work in practice. This project gave me valuable experience in setting up a full-stack application and building a functional backend from scratch.
